# FACTOVA Launcher 패치노트
## 버전 1.8.0 (2025-01-XX)
- 기능: 운영 모드에서 프로그램 시작 시 외부 실행 프로세스 자동 감지
- 기능: 외부 실행된 프로세스도 모니터링하여 종료 시 런처 자동 복원
- 기능: 우클릭 메뉴에 "백업 없이 바로 실행" 옵션 추가 (굵게 표시)
- 개선: 외부 실행 시 자동 백업 건너뛰기 (백업 대상 폴더 알 수 없음)
- 개선: 프로세스 감지를 1초 간격으로 지속 모니터링
- 수정: 외부 실행 프로세스 종료 시 null 참조 에러 수정

## 버전 1.7.0 (2025-01-XX)
- 기능: 버튼 폰트 크기 조절 기능 추가 (8-72pt)
- 기능: URL 커스터마이징 기능 (우클릭 메뉴)
- 개선: 자동 백업 On/Off 설정 추가

## 버전 1.3.0 (2025-10-23)
- 기능: 운영 모드에서 우클릭하여 백업 처리함

## 버전 1.2.0 (2025-09-29)
- 기능: 설정 탭에 패치노트 뷰어 추가
- 기능: 설정 탭에 현재 적용된 설정 파일의 어셈블리 정보 표시 기능 추가
## 버전 1.1.0 (2025-09-26)
- 기능: GMES 1.0 모드 추가 (개발자 모드 전용)
- 수정: GMES 1.0 모드에서는 일반 모드로 전환 불가
- 수정: GMES 버전에 따라 다른 설정 파일 이름 적용 (`LGE.SFC.MainFrame.exe.config`)
- 수정: `ConfigPath`가 현재 사용자 계정에 맞게 동적으로 수정되도록 변경
## 버전 1.0.0 (2025-09-24)
- 최초 릴리즈
- 기능: GMES 2.0 설정 불러오기/백업
- 기능: 개발자 모드 (Ctrl + 타이틀 클릭)
- 기능: 다국어 지원 (한국어/영어)
